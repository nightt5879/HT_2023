/* Holtek HT32 TinyProgrammer Maker III - Version 1.00                                                      */
/*----------------------------------------------------------------------------------------------------------*/
  Release Date: 2015-11-10

  HT32 TinyProgrammer Maker III is a PC-Based graph user interface for generate single EXE file which
  including TinyProgrammer III and application image. The TinyProgrammer III communicates with ISP/IAP
  firmware via USB interface and then uses the pre-define protocol/commands to apply flash operations.

    Path:                                                  Descriptions:
    ----------------------------------------------------------------------------------------------------------
    {INSTALL_PATH}\TinyProgrammerMakerIII.exe              HT32 TinyProgrammer Maker III
    {INSTALL_PATH}\HTEncrypt32.exe                         HT32 Encryption Tool
    {INSTALL_PATH}\default.ini                             INI file of default setting
    {INSTALL_PATH}\setting.ini                             INI file of current setting
    {INSTALL_PATH}\HTEncrypt32.ini                         INI file of current setting for HTEncrypt32
